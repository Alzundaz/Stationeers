# Dual Tower

## Infos

## Income Script
Script : [IncomeControl](/Scripts/DTTC/IncomeControl)

- D0 : Vent Memory
- D1 : Gas Tank
- D2 : Exterior Sensor
- D3 : Gas Dump

## Pressure Script
Script : [PressureControl](/Scripts/DTTC/PressureControl)

- D0 : Hot Income Pump
- D1 : Hot Dump Pump
- D2 : Hot Sensor
- D3 : Cold Income Pump
- D4 : Cold Dump Pump
- D5 : Cold Sensor

## Transfer Script
Script : [TransferControl](/Scripts/DTTC/TransferControl)

- D0 : Coolers Memory
- D1 : Hot Sensor
- D2 : Cold Sensor
